= ベンダー固有のリファレンス
:hardbreaks:
:allow-uri-read: 


[role="lead"]
このセクションのトピックでは、ベンダー固有のリファレンス情報を提供します。ほとんどの場合、データ コレクターの構成は簡単です。場合によっては、データ コレクターを適切に構成するために追加の情報やコマンドが必要になることがあります。

左側のメニューで *ベンダー* をクリックすると、そのデータ コレクターの情報が表示されます。
