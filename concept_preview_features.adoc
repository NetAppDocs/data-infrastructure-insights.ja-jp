= プレビュー機能
:hardbreaks:
:allow-uri-read: 


[role="lead"]
Data Infrastructure Insights、新しい機能と機能を _プレビュー_ 形式で紹介します。プレビュー機能は、制限された環境または制限された容量で利用可能になります。

これらの機能の 1 つ以上をプレビューすることにご興味がある場合は、 NetApp の営業チームに詳細をお問い合わせください。
