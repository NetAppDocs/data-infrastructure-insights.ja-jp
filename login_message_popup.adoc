= ログインメッセージの設定
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ログイン時にすべてのユーザーにポップアップメッセージとして表示するように設定できます。次回のログイン時のみ表示するか、以降のすべてのログイン時に表示するかを選択できます。

管理者は、[管理] > [通知] の [ログインバナー] タブでメッセージを設定できます。

このメッセージは基本的なMarkdown形式に対応しているため、太字、斜体、表などの書式設定を適用できます。

ユーザーが次回ログイン時のみメッセージを表示するか、以降のすべてのログイン時にメッセージを表示するかを選択してください。

image:login_banner_settings.png["ログインバナー設定"]

ログインメッセージを閉じるには、ユーザーがログインメッセージを承認する必要があることに注意してください。一度閉じると、それ以降のポップアップ（新機能の更新、利用規約の更新など）が表示されます。
